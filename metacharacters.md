# Metacharacters

Die folgenden Zeichen sind sogenannte Metacharacters:&#x20;

```javascript
^$.*+?=<>!:|\/()[]{}
```

&#x20;^    Start eines String

&#x20;$    Ende eines Strings

&#x20;.      Beliebiges Zeichen

&#x20;\*     0-mal oder mehrere

\+      1-mal oder mehrere

&#x20;?      0-mal oder 1-mal

( )     Gruppe

&#x20;\[ ]     Set

&#x20;{ }     Quantifier

&#x20;\        Escape Character
